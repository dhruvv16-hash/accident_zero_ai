This runbook operates **all features at least once**: original ML training, API endpoints, frontend charts, Excel batch flow, image risk flow, Excel-vs-image individual frontend analysis, advanced fusion extension, simplified single-visual fusion, and timestamped evidence generation.

### 4) Original Frontend Features (use once each)
1. Manual risk prediction:
- Fill safety inputs and click `Predict Risk`.

2. Excel batch prediction:
- Upload `data\sample_upload.xlsx` (or other `.xlsx`) and click `Predict from Excel`.

3. Correlation matrix:
- Click `Compute Correlation Matrix`.

4. Existing charts visible in UI:

- Model Ensemble Overview
- Risk Distribution (Batch)
- Selected Row Model Comparison

### 5) Image Risk Features (use once each)
1. Single/multi image analysis:
- Upload images in `Image Risk Analysis` and click `Run automatic scene analysis`.

2. Blend controls:
- Click `Add image analysis to main output`.
- Then click `Revert image blend`.

3. Quota-safe behavior:
- If Gemini quota is exceeded, backend now returns fallback result (workflow still continues).
